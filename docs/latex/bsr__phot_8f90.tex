\mbox{\hyperlink{namespacebsr__phot_abb0b503cf184624d5cae3d509d7ef623}{bsr\+\_\+phot.\+inp}} -\/ input parameters

w.\+nnn -\/ the name for file with expansion coefficients for internal region solutions. These data are used to obtain the contibution of different closed channel for given resonanes.
